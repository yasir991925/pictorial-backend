== Building

To launch your tests:
```
./gradlew clean test
```

To package your application:
```
./gradlew clean assemble
```

To run your application:
```
./gradlew clean run
```

To run with docker-compose
```
gradle shadowJar && docker-compose up --build --remove-orphans
```
